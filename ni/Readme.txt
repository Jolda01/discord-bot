Le script python du bot utilise le token du bot qui est une information secret, ce qui signifie on doit l'ajouter comme variable d'environement.

le bot a 2 commands:
$quote: le bot envoie une citation du HIV.

$input: le bot demande une couleur qui ensuite changera l'arriére plan du site web associé avec ce projet.

l'input du bot sera sauvegardé dans C:\\color.txt (sera crée s'il n'existe pas) et ce dernier sera accéde par le script php du site web lors du démarrage du site web.
